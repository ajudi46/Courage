# MVP Scenarios – Decision Training App (Courage System)

This file contains **20 MVP scenarios** across **5 core decision themes**.
Each scenario is fully tagged to enable pattern detection, personalization, and analytics.

---

## Tagging Schema (Used for All Scenarios)

Each scenario includes the following metadata:

- **Scenario ID**: Unique identifier
- **Theme**: Core behavioral pattern being tested
- **Context**: Life area (Office / Personal)
- **Role**: User role assumption
- **Power Dynamic**: Peer / Authority / Subordinate / Self
- **Stakes Level**: Low / Medium / High
- **Tension Axis**: Core trade-off being tested
- **Option A Posture**: Decision posture of swipe left
- **Option B Posture**: Decision posture of swipe right
- **Avoidance Indicator**: Which option represents avoidance
- **Primary Signal Tracked**: What this scenario contributes to
